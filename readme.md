# 1. Install to Os

`sudo apt install ffmpeg`

## 2. install packages (requirements.txt)

`pip install -r requirements.txt`

### NOTE: whisper.ai requires Python >=3.7,<3.11

# Usage

```
python main.py -i <input.wav> -o <output.json>
```
